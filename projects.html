A collection of data-driven solutions showcasing expertise in machine learning, analytics, and visualization. These projects demonstrate problem-solving skills, statistical modeling, and the ability to extract insights from complex datasets to drive informed decisions.

Wild Fire Detection using Deep Learning

This research comprehensively explores various Machine Learning algorithms and methodologies employed in wildfire detection. Through a thorough review of related literature, we analyze the efficacy of different approaches, ranging from deep learning models such as Convolutional Neural Networks (CNNs) to Recurrent Neural Networks (RNNs) and beyond. By synthesizing insights from prior research, we identify trends, strengths, and limitations of existing techniques, laying the groundwork for our proposed solution.

Smart Attendance System

Simple and user friendly face recognition system that uses advanced technologies like Image Processing with python ,machine learning algorithms and PHP API and for front end we have used HTML & CSS .The main objective of invention is in which number of students present during a lecture is observed (or employees),through webcam their image is captured in the system and then the face is compared with the data stored in Database which later stores the information of attendance.

Car Prediction System

A machine learning-based system that predicts car prices using historical data and key features such as brand, model, year, mileage, fuel type, and engine capacity. Utilizing algorithms like Linear Regression, Random Forest, and XGBoost, the model analyzes market trends and provides accurate price estimates. The system helps buyers and sellers make data-driven decisions in the automotive market.

Movies Recommentation System

Designed a collaborative filtering-based recommendation engine for personalized content suggestions.Optimized matrix factorization techniques for better user experience and retention.

IPL Data Visualization Tableau

This project analyzes Indian Premier League (IPL) data using Tableau to uncover key insights on team performance, player statistics, match outcomes, and historical trends. Interactive dashboards showcase metrics like top run-scorers, highest wicket-takers, win/loss ratios, and venue-based performance, enabling users to explore patterns and make data-driven decisions.
